
# Library Management System (Core Java + MySQL)

## Steps to Run

1. Make sure MySQL is installed and running.
2. Import `library.sql` into your MySQL server.
3. Update `DBConnection.java` with your MySQL username and password.
4. Add MySQL JDBC driver (JAR) to the `lib/` folder.
5. Open this project in VS Code.
6. Compile and run `Main.java`.
